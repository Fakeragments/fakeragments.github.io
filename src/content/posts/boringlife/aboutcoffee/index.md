## 橘香茉莉SOE
1. 美式

|       |              |
|------|------|
| **烘培度** | 中浅 |
| **机器** | 可数S1 |
| **研磨器具** | 可数磨豆机 |
| **粉量** | 18g |
| **出液** | 28g |
| **温度** | 93℃ |
| **萃取时间** | 30s 算预浸泡 |
| **压力** | 稳11bar |

![alt text](image-4.png)
冰100g 水100g，兑浓缩液，果酸味适中，是恰到好处的酸，酸后的甜感像是大杂烩吧，如果硬要往标注的口味上靠的话，是有类似于橘子的味道，茉莉花这块，，当就当有点吧，作为中浅烘来说，这个酸度刚刚好，比之前摩卡壶煮出来的酸度好很多。
